# Laravel Cycle ORM Adapter

Unlock the full potential of [Domain-Driven Design](https://en.wikipedia.org/wiki/Domain-driven_design) in your Laravel projects with the [wayofdev/laravel-cycle-orm-adapter](https://github.com/wayofdev/laravel-cycle-orm-adapter) â€” the adapter package that seamlessly integrates the capabilities of [CycleORM](https://cycle-orm.dev) â€” DataMapper ORM into [Laravel Framework](https://laravel.com).

This adapter bridges the gap between Laravel's rapid development capabilities and Cycle ORM's Data Mapper features, enabling you to craft complex, domain-centric applications, with separation of concerns, and a clear distinction between the domain model and the persistence layer.
